What is SQLite?


Is an in-process library that implements a self contained, serverless, SQL database engine. It is free for commercial and private use.


What is SQL?


SQL stands for Structured Query Language. It's a special purpose programming language, designed for manipulating data in a relational database management system. Founded by Edgar F. Codd's in the 70's, it is often described as a declarative language but it also includes procedural elements as well.


What are Relational Databases?


Relational databases are the most common type of database. They database contains one or more tables, where each table consists of 0 or more rows of data. The data for each row is organized into units called columns.




The tables in a database will often have links/relationships between them. 'one-to-one' and 'one-to-many' are the most common. You can learn more here SQL
